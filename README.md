# LICHEN: A LIveCoding Historical ENvironment for the GRASS Language.

LICHEN (LIveCoding Historical ENvironment) is a JavaScript implementation of the pioneering GRASS (GRAphics Symbiosis System) livecoding language developed by Tom DiFanti starting in 1973, in its historical context as a scriptable video source for the hardware analog effects system called the Sandin Image Processor, by Dan Sandin and Phil Morton. The Sandin hardware is also entirely recreated by LICHEN, using a JS port of an emulation project by <a href="https://github.com/amandalong">@amandalong</a>.
